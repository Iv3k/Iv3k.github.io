Web Apps

Reactivities Social Network

Social Network for reactivities. You can create reactivities, edit them, and delete them. Additionally, you can add other users, chat with them,view their activities and also upload photos. The main focus is on clean code architecture and using the mediator pattern.

Technology: C# .Net Core, React, TypeScript, MobX, SignalR.

Joke Generator

You can select the category of the joke, search using different words, and specify the language of the joke. After that, you can generate your joke. I used a joke API for all jokes and accessed them via Axios. No offense intended for anyone; it was for fun and practice.

Books Record

Web development project for exercise. You can view previous books, enter new ones, and delete books. Book covers are received from the API. I used JavaScript for the frontend, Express for the backend, and PostgreSQL for storing all information locally.
